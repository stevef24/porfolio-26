## Learnings

1. **Instrument Serif only has weight 400** - Hierarchy is achieved through size, not weight. All `font-weight: 500` or `600` references for display font render as 400.

2. **Oatmeal uses hex values** - Previous design used OKLCH. Oatmeal spec uses hex for predictable cross-browser rendering.

3. **Key color palette:**
   | Token | Light | Dark |
   |-------|-------|------|
   | background | `#f8f6ef` | `#0e0e0c` |
   | foreground | `#19170f` | `#f8f6ef` |
   | primary | `#566240` | `#6f7c5a` |
   | muted | `#e8e5da` | `#1e1e1b` |

4. **Button size change is significant** - Default went from h-8 (32px) to h-12 (48px). This affects layout throughout the app.

5. **Typography classes added:**
   - `.text-swiss-body-lg` - 20px lead paragraphs
   - `.text-swiss-section` - 32-48px section headings

### What Failed

### Key Decisions
- **Decision:** Replace all OKLCH values with hex
  - Alternatives: Keep OKLCH for some tokens
  - Reason: Oatmeal spec uses hex; consistency matters more than color-space precision

- **Decision:** Use pill shape (rounded-full) for buttons
  - Alternatives: Keep rounded-sm or use rounded-lg
  - Reason: Oatmeal spec explicitly requires pill buttons for modern aesthetic
